FAQ1

  • 处理器：ARM926EJ-S内核，主频最高454Mh
  • 动态内存： DDR SDRAM 64MB
  • 固态存储： MicroSD卡（最高支持32GB）

FAQ2

  • Bootloader: u-boot 或 niuboot (small size)
  • Linux kernel version: 2.6.31
  • Distribution: Debian-arm, Android, Firefox OS, Openwrt, Federa-arm
  • RTOS: eCos, free-rtos
